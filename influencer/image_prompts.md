## Tips for Best Results

- **For DALL-E 3:** Paste the full prompt as-is. DALL-E 3 handles long descriptive prompts well.
- **For Stable Diffusion XL:** Consider splitting into a positive prompt and adding a negative prompt such as: `cartoon, anime, illustration, ugly, deformed, watermark, blurry, low quality, extra limbs`
- **Consistency tip:** Using the same base subject string across all generations helps maintain a coherent visual identity for the character.
- **LoRA/fine-tuning:** For maximum consistency, train a character LoRA on 20–30 generated images and apply it across all prompts.
